						About SunnyLand


How to download the game:

Step 1: Download this repository.
Step 2: Use the password provided in the submission application to extract the zip file contents/main game.
Step 3: Run the "SunnyLand.exe" file to play the game (no installation is required).


Controls:

Arrow Keys- for movement
Space bar - For jumping


About Game:

Your end goal is to move right and reach HOME crossing all the obstacles to complete the game.
If you jump on the frog you get 1 or 2 points (randomly selected).
You may choose to skip over the frog to move on; but you won't earn any points.
If you touch the frog you loose your life and the game ends.


(Please download WinRAR for extracting the zip file)
